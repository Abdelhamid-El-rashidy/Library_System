Role-Based Access

Every user gets the right tools for their needs.

Reader

Regular User

Browse, search, borrow, and return books. Track your borrowed items and due dates from your personal dashboard.

  • Browse full catalog
  • Search by title, author, category
  • Borrow books for 14 days
  • Track due dates
  • Return borrowed books
Admin

Administrator

Manage the library collection. Add, edit, and remove books. Upload cover images and oversee all borrowed books.

  • All Reader permissions
  • Add new books
  • Edit existing books
  • Delete books
  • View all borrowed books
  • Mark any book as returned
Superuser

Superuser

Full system control. Everything an admin can do, plus the ability to create and remove administrator accounts.

  • All Administrator permissions
  • Create admin accounts
  • Remove admin accounts
  • Full system oversight
